Что изучают на программировании?

Первый уровень обучаются на платформе Scratch, они создают свои игры, анимации, презентации, живые картинки, на первых занятиях дети ознакамливаются с компьютером, как правильно скачивать, как устанавливать программы. На втором уровне App Inventor дети создают приложения для смартфонов Android Третий уровень Веб-программирование, HTML CSS, на этом уровне дети создают свои сайты.
